Welcome to my page!

Hi! My name is Maaria Wahlström, I live in Vantaa, Finland and I intend to become a professional programmer. My plan is to learn to code, then code some cool things, then get a job where I can learn more coding and make more cool things.

My coding experience

My first foray into programming was a language technology course where we learned to use Python as a tool for analyzing text samples. I loved the puzzle-like nature of the exercises - tweaking the code until you get the desired result. I felt the same drive when learning to use R in statistical analysis on a psychometrics course. I decided to follow this passion and started my programming studies in July 2019.

Since then I've done projects with Java, using Maven or Gradle for dependency management, and more recently with Python using Poetry. I've had quite a bit of practice with Git and all my projects can be found in GitHub, some of which also have GitHub Actions facilitating continuous integration. I've also learned the basics of SQL, and used a PostgreSQL database with a couple of Heroku apps. Other technologies that I've dabbled with are Flask, JavaScript, NodeJS and React.

Most of my projects have been individual works, but at the end of 2021 I got to participate in a group project. The web app we made together was very small, but the emphasis was on learning the process of software engineering as a team, and implementing Scrum.
