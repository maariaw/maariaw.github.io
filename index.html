My coding experience

My first foray into programming was a language technology course where we learned to use Python as a tool for analyzing text samples. I loved the puzzle-like nature of the excercises - tweaking the code until you get the desired result. I felt the same drive when learning to use R in statistical analysis on a psychometrics course. I decided to follow this passion and started my programming studies in July 2019. Since then I've learned the basics of Java, SQL, HTML, CSS and Git. Right now I'm brushing off my Python and getting into JavaScript.
